系统命令模板 (支持 {FLAG} 占位符)
💡 使用说明:
  • 使用 {FLAG} 占位符，服务端会自动替换为实际的flag值
  • 支持任意shell命令：curl、wget或自定义脚本
  • 命令通过 sh -c 执行，请确保命令格式正确
  • 示例1 (curl POST): curl http://platform.com/submit -d '{"flag":"{FLAG}"}'
  • 示例2 (curl GET): curl "http://platform.com/submit?flag={FLAG}"
  • 示例3 (wget): wget -O- --post-data='flag={FLAG}' http://platform.com/submit
  • 示例4 (脚本): /path/to/submit.sh "{FLAG}"
  • 注意：服务端已有90秒延迟，命令中通常不需要再加sleep
